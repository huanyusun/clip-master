功能说明

核心功能

  • 剪贴板历史记录 - 自动保存复制的文本、图片、文件和链接等内容
  • 内容分类 - 按类型和自定义分类组织剪贴板内容
  • 快速搜索 - 通过关键词快速查找历史剪贴板内容
  • 固定重要内容 - 将常用内容固定在顶部方便访问
  • 快捷键支持 - 通过键盘快捷键快速访问和操作
  • 菜单栏快速访问 - 通过macOS菜单栏图标快速访问最近内容
  • 隐私保护 - 自动过滤敏感信息，支持隐私模式
  • 内容预览 - 支持文本、图片、代码等多种格式的预览
  • 自定义标签 - 为剪贴板内容添加自定义标签和分类
  • 数据同步 - 可选择通过iCloud在多设备间同步剪贴板历史

技术实现

  • 使用NSUserDefaults或专用配置文件存储用户设置
  • 实现全局快捷键支持，使用Carbon Hot Key API
  • 使用NSStatusItem实现菜单栏快速访问功能
  • 实现AES-256加密算法保护敏感数据
  • 采用响应式设计确保在不同屏幕尺寸下的良好体验
  • 使用SwiftUI动画提供流畅的交互体验
